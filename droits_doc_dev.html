Fonctionnement

Utilisation

  • En PHP : $Specific->getRight('nom_du_droit')
  • En js (version 2015) $scope.specific_rights.nom_du_droit
  • Dans un template SMARTY : 'nom_du_droit hasRight

Les droits et les versions

S'il est nécessaire d'activer un droit pour toutes les versions de THR il faut que le droit soit déclaré dans le PHP. Au contraire si ce dernier doit être utilisé uniquement sur une seule version alors il est préférable de le déclarer dans le js.

Base de données

voici la procédure a suivre :

Pour ajouter un droit à un camping il faut ajouter une requête dans le fichier sql/scripts/"version"/update.sql. REPLACE INTO `thelisresa`.`RIGHTS` (`Idclicamp`, `key`, `right`) VALUES ('xxx', 'nom_de_la_variable', 'X');
